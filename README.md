# spring-boot-vuejs

## Build
* `git clone https://github.com/hosekim/spring-boot-vuejs.git`<br>
* `cd spring-boot-vuejs`<br>
* `mvn install`<br>
* `mvn --projects be spring-boot:run`<br>

## Test
* `localhost:8088`
